# AOK Hisobot — Sayt versiyasi v3.0

O'quv markaz uchun hisobot tizimi. Sayt + telefon ilovasi (PWA).

## 📁 Fayllar

```
index.html      — asosiy sayt
manifest.json   — PWA sozlamalari (telefon app)
sw.js           — service worker (offline ishlash)
icon-192.png    — ikonka (kichik)
icon-512.png    — ikonka (katta)
```

## 🖥️ Kompyuterda sinab ko'rish

`index.html` ni brauzerда oching (2 marta bosing).

## 🌐 Internetga joylash (GitHub Pages)

1. github.com da hisob oching (bepul)
2. Yangi repository yarating: `aok-hisobot`
3. Barcha fayllarni yuklang
4. Settings → Pages → Branch: main → Save
5. Sayt tayyor: `https://NOMINGIZ.github.io/aok-hisobot`

## 📱 Telefonga o'rnatish (PWA)

1. Telefonда saytni Chrome/Safari da oching
2. Menyu → "Bosh ekranga qo'shish" (Add to Home Screen)
3. Telefon ekranida AOK ikonkasi paydo bo'ladi
4. Ilova kabi ochiladi!
